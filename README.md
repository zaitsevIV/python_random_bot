# telegram_random_bot @common_random_bot
This bot based on pyTelegramBotAPI library

## Description of the program:

#### 1. /random_number
generate a random number in your range

#### 2. /random_dice_number 
roll the dice and print the number

#### 3. /random_password - 
create a password
